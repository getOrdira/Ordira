🧪 Remember Me Functionality Test

📋 Test Instructions:

  1. Open your browser's Developer Tools (F12)
  2. Go to the Application/Storage tab
  3. Look for Cookies section
  4. Navigate to your localhost domain
  5. Login with "Remember Me" checked and unchecked
  6. Check if remember_token cookie appears when checked

🔍 What to Look For:

  • With Remember Me: Should see remember_token cookie with 30-day expiration
  • Without Remember Me: Should NOT see remember_token cookie
  • Regular auth cookies: Should see auth-token and refresh-token in both cases

🌐 Test URLs:

Desktop Login: http://localhost:3000/auth/login

Mobile Login: http://localhost:3000/auth/login/mobile

🔧 Manual Test Steps:

  1. Go to login page
  2. Enter valid credentials
  3. Test 1: Leave "Remember Me" unchecked → Login → Check cookies
  4. Test 2: Check "Remember Me" → Login → Check cookies
  5. Compare the cookie results

📊 Expected Results:

✅ Remember Me CHECKED:
- auth-token: [present]
- refresh-token: [present] 
- remember_token: [present with 30-day expiration]

✅ Remember Me UNCHECKED:
- auth-token: [present]
- refresh-token: [present]
- remember_token: [NOT present]

🐛 Troubleshooting:

  • No cookies at all: Check if login is successful
  • remember_token missing when checked: Check backend logs for errors
  • Cookie not secure: Check if running in production mode
